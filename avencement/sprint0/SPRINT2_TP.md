# Sprint 0 – Séance de TP 1

**Projet : LO3BA (ACL-ISN 2025 – Mines Rabat)**

---

## 1. Backlog Sprint 1

### Objectif global

Mettre en place la première version du jeu **LO3BA** avec un menu principal fonctionnel et la navigation vers la scène de jeu.

### User Stories

| En tant que... | Je veux...                                   | Afin de...                          |
| -------------- | -------------------------------------------- | ----------------------------------- |
| Joueur         | Voir un écran d’accueil avec le titre du jeu | Identifier facilement le jeu        | 
| Joueur         | Cliquer sur “START GAME” pour commencer      | Accéder à la scène du jeu           | 
| Joueur         | Avoir un bouton “EXIT”                       | Pouvoir quitter le jeu simplement   | 
| Développeur    | Appliquer un thème visuel rétro              | Donner une identité visuelle au jeu |
